## Proyecto simple de app Mouse-Follower que hace un seguimiento del mouse en patalla:
1- Crear el proyecto en React Vite:
```bash
npm create vite@latest
```

2- Instalar las librerias dentro de la carpeta mouse-follower:
```bash
npm install
```
